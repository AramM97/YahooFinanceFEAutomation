# Yahoo Finance FE Automation Testing Project

## Overview

This project focuses on automating Front-End testing for Yahoo Finance using Python, Selenium, and Selenium Grid. The primary functionalities covered include creating a portfolio, adding stocks to the portfolio, downloading stock history, checking stock profiles, and retrieving information about the most active stocks.

## Prerequisites

Make sure you have the following installed before running the tests:

- Python 
- Selenium WebDriver
- Selenium Grid

## Test Descriptions
1. Portfolio Creation
Test the functionality to create a new portfolio on Yahoo Finance.

2. Add Stock to Portfolio
Test the capability to add a stock to an existing portfolio.

3. Download Stock History
Verify the ability to download historical data for a specific stock.

4. Check Stock Profile
Ensure accurate retrieval of stock information and profile.

5. Get Most Active Stocks
Test the functionality to retrieve information about the most active stocks.
